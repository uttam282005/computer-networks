# Applications (Section 1.1)

- Computer networks enable resource sharing, communication, and distributed applications.
- The Internet is a global network connecting millions of devices.
- Applications include email, web browsing, file transfer, remote login, and multimedia streaming.
- The web is a major application but not all network applications are part of the web (e.g., streaming, file transfer).
- Networks support both human-to-human and machine-to-machine communication.
- Application requirements (e.g., latency, bandwidth, reliability) influence network design.
